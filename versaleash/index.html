VersaLeash

VersaLeash is a Minecraft plugin aimed at all things lead/leash related. For this plugin, it allows leashing of more entities than what vanilla allows, still some limits on what entities can be leashed. The plugin also supports an anti-breaking feature; and there is yet another feature for allowing entities that are leashed to be immune to damage so long as they are on a lead, this is optional.
