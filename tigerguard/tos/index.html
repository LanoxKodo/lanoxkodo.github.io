2. Description of TigerGuard

TigerGuard is an open source bot provided as a service for use exclusively within Discord. It offers features such as; in-call audio features, xp-leveling systems, reaction role handling, color roles, level associated guild roles, and polls, dice rolls, custom embed creation. All of which has administrative checks if the logic associated needs admin-like permissions for setting up and/or configuring for use within a server.

The Bot does not collect nor process any personal data beyond what is needed to perform functions. It is written to keep data as agnostic as possible and thus no data about you outside of Discord is ever reached by the Bot, your Discord UUID at most is the only thing that is held about you.
