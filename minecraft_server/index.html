Twilight Isles MC Server

Cross-play these days has become a big point in gaming, Minecraft is no stranger to people trying to play with one another. Minecraft is built with two main editions, those are Minecraft Java Edition and Minecraft Bedrock Edition, these natively cannot play with one another and are kept to their own selves for clients to play with. But through dedication and support of other individuals who too sought to bridge that gap have made it possible for a single Minecraft Java Edition server to allow Bedrock users to join through the use of a single plugin!

I have been hosting servers for Minecraft since 2016 and am excited to see that I can extend this to a more official longterm server setup that people on any edition of Minecraft can play on that they wish.
Follow the steps below and you will be on the server in no time!

How to Join

In your Minecraft client, enter one of the following IPs followed by the related port number for your edition.
Use the '.dyn' IP if you use an OpenNIC DNS. If you wish to use this option, see Advanced DNS Setup.

Examples:
Java: LanoxKodoDev.fr.to:36676
Bedrock: LanoxKodoDev.fr.to:19132

Valid IP Domains:

LanoxKodoDev.fr.to
LanoxKodoDev.dyn

Java Port

36676

Bedrock Port

19132

Server Plugins and Usable Commands

Because the server is setup for both Java and Bedrock users, any modifications are limited to server-side changes which means only plugins and data packs. The server currently has many plugins in active use at this time which I often let users give me ideas they'd like to see added and then we have a vote session hosted for a moment in time, usually a week, to allow people to have a say with what might be added.

If you would like to see the list of current plugins currently added, you may do so by clicking here: Our Server Plugins

Advanced DNS Setup

To join the server using an OpenNIC DNS, you'll need to configure your device to use a DNS listed on the OpenNIC server list, located here. Check your device settings on how to change your DNS settings. Once your DNS settings have been changed, you will be able to access the server using the .dyn TLD.
