Kodo Alt Crafting Plugin

The Kodo Alt Crafting plugin is a simple plugin for non-vanilla recipes. Currently it supports doing the following in a Furnace:
-Convert concrete powder blocks of a color into stained glass blocks of the same color
-Convert concrete blocks of a color into concrete powder blocks of the same color.

Both of these recipes work for all 16 color variants so you can use whichever you like and can expect that whatever color material you put in to smelt will output the relevant end result like seen here above!
Additionally there are some crafting recipes that are added to players Knowledge book upon entering the server, they are alternative recipes such as converting block of Quartz back into 4 pieces of quartz.

Design Idea Source

The idea for this plugin was to allow players to originally make glass blocks of colored variants from the respective colored concrete powder block similar to making regular glass from sand blocks. Further on, development led to also allowing users to convert concrete blocks of any color back into their powder version of the same color.
