KodoAltHandlers

What a name, anywho. KodoAltHandlers is a Minecraft plugin aimed at things that could very much be their own plugins but don't belong in other projects I currently have. The plugin itself is the successor of another plugin I made which is now depreciated in favor of this one. The theme for these generally is Alternate-Handling, hence the weird name. Given this design choice, it is fully modular so as I update it, if a new additional feature is added, it is completely optional by enabling/disabling in the config.
