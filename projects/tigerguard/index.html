TigerGuard

TigerGuard is a mostly administrative discord bot that is in Beta but stable for all intended purposes. It supports typical management or role provisioning with custom role support, a leveling system, and a few other things.

Everything is Opt-In with this bot, to make this bot work for you, you must specify through the bot administative command what you would like to set up.
